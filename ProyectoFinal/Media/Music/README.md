# Carpeta de Música de las Historias

Coloca aquí todos los archivos de sonido/música (.mp3, .wav, .m4a) que referencies en tus archivos `.json` de historia.

Por ejemplo, si en tu JSON pones:
`"musicTrack": "suspenso_bosque"`

Debes colocar una canción llamada `suspenso_bosque.mp3` o `suspenso_bosque.wav` en esta carpeta.
