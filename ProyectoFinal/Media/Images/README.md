# Carpeta de Imágenes de las Historias

Coloca aquí todos los archivos de imagen (.png, .jpg, .jpeg) que referencies en tus archivos `.json` de historia.

Por ejemplo, si en tu JSON pones:
`"backgroundImage": "bosque_oscuro"`

Debes colocar una imagen llamada `bosque_oscuro.png` o `bosque_oscuro.jpg` en esta carpeta.
